# CIBUS

Web application for managing cooking recipes. Calculates the macronutrients of each recipe, generates a shopping list with the estimated price per ingredient and the total cost. Nutritional data is fetched from Open Food Facts.

## Stack

- **Backend:** Java 21 + Spring Boot
- **Frontend:** HTML, CSS, JavaScript vanilla
- **Database:** MySQL
- **External API:** Open Food Facts
- **Security:** Spring Security + JWT + BCrypt
